# Gestión de Historiales Clínicos Temporales – POO en Python

Este repositorio contiene un ejemplo académico desarrollado en Python que ilustra el uso de **constructores** (`__init__`) y **destructores** (`__del__`) dentro del paradigma de la programación orientada a objetos (POO). La implementación está contextualizada en el ámbito de la **atención en urgencias y emergencias**, simulando la apertura y cierre seguro de historiales clínicos temporales durante la atención de pacientes.

## Objetivo

Demostrar el ciclo de vida de los objetos en Python mediante una aplicación relevante para las ciencias de la salud, resaltando buenas prácticas de programación y la importancia de la gestión responsable de recursos en entornos clínicos.

## Características

- Clase `HistorialClinicoTemporal` que representa un expediente médico temporal.
- Constructor que inicializa datos del paciente (nombre, ID y nivel de triaje).
- Destructor que simula el cierre controlado del historial y liberación de recursos.
- Mensajes en consola que permiten visualizar claramente la activación de ambos métodos especiales.
- Código comentado, legible y estructurado según estándares académicos.

## Requisitos

- Python 3.x
- Ninguna dependencia externa

## Ejecución

1. Clonar o descargar el repositorio.
2. Abrir una terminal en la carpeta del proyecto.
3. Ejecutar el script:
